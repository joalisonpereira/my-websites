O que faz um Desenvolvedor Freelancer?

Um Freelancer, também conhecido popularmente no Brasil pelas expressões ou gírias freela ou frila, é o termo inglês para denominar o profissional autónomo que se autoemprega em diferentes empresas ou, ainda, guia seus trabalhos por projetos, captando e atendendo seus clientes de forma independente.

Um desenvolvedor web atende clientes de forma remota, seja na criação de sites,blogs e e-commerce's tornando-se o processo o mais cômodo possível para todas as partes.

Algumas das Minhas Habilidades

HTML & CSS

HTML é uma linguagem de marcação utilizada na construção de páginas para Web.

CSS é um simples mecanismo para adicionar estilo a um documento web, ele cria um link para uma página que contém os estilos.

PHP

PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.

LARAVEL

Laravel é um framework PHP livre e open-source criado por Taylor B. Otwell para o desenvolvimento de sistemas web que utilizam o padrão MVC.

Bootstrap

Bootstrap é um framework front-end que facilita a vida dos desenvolvedores web a criar sites com tecnologia mobile (responsivo) sem ter que digitar uma linha de CSS para “fazer e acontecer”.



  • DESIGN RESPONSIVO


    Design Responsivo é aquele que consegue se adaptar a diferentes telas e quando afirmo que ele se adapta não estou dizendo que ele simplesmente funciona, um projeto responsivo se reorganiza da melhor forma para apresentar seu conteúdo de acordo com o dispositivo do usuário.

  • SEARCH ENGINE OPTIMIZATION


    O SEO (Search Engine Optimization), também conhecido como otimização de sites, é rapidamente definido como uma forma de aumentar os acessos do seu site através de um conjunto de técnicas e estratégias que permitem que um site melhore seu posicionamento nos resultados orgânicos dos mecanismos de busca.
